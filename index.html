Web ontwikkeling

Noem het niet webdesign want het is zoveel meer. Design en programatie. Ook heel wat ervaring met CMS systemen zoals Drupal en Wordpress. Dankzij een extra intensieve opleiding bij CVO te Heusden zolder heb ik heel wat te bieden.

Grafisch ontwerp

2d en 3d illustraties, ook hier kan ik heel wat mooie dingen bieden. Met een dankjewel aan alle kansen die ik reeds gekregen heb van toch wel bekende merken, reclamebureau's en diverse andere opdrachtgevers.

Animatie

Animatie is ook een passie waar ik steeds mijn kennis in blijf verrijken. Ook hier heb ik het geluk gehad te mogen meewerken met grote namen.

Een voorwoordje over deze site

Deze site is een eindwerk dat ik tijdens mijn opleiding als web ontwikkelaar heb mogen maken. Er zit dus nog niet alles in. Dit eindwerk was voor het vak Interface lab deel 2. De opdracht bestond uit een statische website! Zelfs het responsieve navigatie menu werkt volledig op css.

Handige tools

PHP of css frameworks zoals Bootstrap of Foundation mochten voor dit bepaald eindwerk niet gebruikt worden. Dankzij pre-processors zoals Jade en SASS bleef het toch een mooie uitdaging.
